# 📖 서비스 소개
이 프로젝트는 망고플레이트 회원에게는 종류별 맛집 검색 및 후기작성 기능을, 사업자에게는 음식점 등록 및 관리를 할 수 있도록 구현한 프로젝트입니다.
   <br/>
   <br/>
   <br/>
## 💻 개발 환경
- **Java** : JDK 11
- **Editor** : Eclipse IDE
- **Database** : Oracle SQL Developer
- **Output Format** : Console
<br/>
<br>
